# EasySec - English
## Introduction
EasySec is a repository that includes a variety of roles, documentation, and tools designed to help small and medium-sized enterprises (SMEs) implement security measures within their own organizations, as well as promoting penetration testing and red teaming exercises. This file will serve as an index to make it easier to review and analyze the available tools and documentation.
